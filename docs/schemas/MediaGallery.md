# MediaGallery
An array of media object stream IDs.

## Usage
See [definition](../definitions/MediaGallery.md).

| ✅                                                              | ✅                                 |

## Examples
### Single Object
```json
[
 "ceramic://kjzl6cwe1j.....bbttt3z6ika5v6"
]
```
### Multiple Objects
```json
[
 "ceramic://kjzl6cwe1j.....bbttt3z6ika5v6",
 "ceramic://kjzl6cwe1j.....bbttt3z6ika5v6"
]
```
